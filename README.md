# seat-system
1.实现的效果：

    　　学生

        首次登录载入数据，再次登陆载入上次登录的缓存。

        点击座位选座，然后更新缓存。在”我的“页面会显示刚刚预约座位的时间段和地点

      2.       班级

        和学生账号不同的是可以选择时间段，预约整间教室

      3.       管理员

        选择相应教学楼、楼层、座位，将”有人“座位更改成”无人“座位，释放座位
        更改教室状态为不可用。
        新增管理员账号（输入新账号和密码）
